Perfis de Usuário
================

O sistema possui três perfis principais:

Administrador
------------

**Permissões:**
* Acesso total ao sistema
* Gerenciar usuários e grupos
* Configurar tipos de documentos
* Visualizar relatórios completos
* Configurar fluxos de aprovação

**Responsabilidades:**
* Configuração inicial do sistema
* Manutenção de usuários
* Monitoramento geral
* Backup e segurança

Aprovador
---------

**Permissões:**
* Aprovar documentos do seu setor
* Visualizar relatórios do setor
* Gerenciar não conformidades
* Aprovar planos de ação

**Responsabilidades:**
* Revisão técnica de documentos
* Aprovação dentro dos prazos
* Feedback para melhorias
* Acompanhamento de indicadores

Usuário
-------

**Permissões:**
* Criar e editar documentos
* Visualizar documentos aprovados
* Registrar não conformidades
* Consultar equipamentos

**Responsabilidades:**
* Criação de conteúdo técnico
* Seguimento de procedimentos
* Reporte de problemas
* Manutenção de dados

Primeiros Passos
===============

Login no Sistema
----------------

1. Acesse a URL do sistema
2. Digite seu usuário e senha
3. Clique em "Entrar"

.. note::
   Se esqueceu sua senha, use a opção "Esqueci minha senha"

Dashboard Principal
-----------------

Após o login, você verá:

* **Resumo de atividades**: Documentos, aprovações, NCs
* **Notificações**: Itens que requerem sua atenção
* **Atalhos rápidos**: Acesso direto às funções mais usadas
* **Indicadores**: KPIs do seu setor

Navegação
---------

O sistema utiliza uma barra lateral com:

* **Dashboard**: Página inicial
* **Documentos**: Gestão documental
* **Aprovações**: Itens para aprovar (se aplicável)
* **Qualidade**: NCs e auditorias
* **Equipamentos**: Controle de equipamentos
* **Relatórios**: Análises e gráficos
* **Configurações**: Administração (se aplicável)

Como Criar um Documento
======================

Passo a Passo
------------

1. **Acesse o menu "Documentos"**
2. **Clique em "Criar Documento"**
3. **Selecione o tipo de documento:**
   * Política
   * Procedimento
   * Instrução de Trabalho
   * Formulário
   * Manual

4. **Preencha as informações básicas:**
   * Título do documento
   * Código (gerado automaticamente)
   * Descrição
   * Setor responsável

5. **Use o editor para criar o conteúdo:**
   * Template automático carregado
   * Formatação rica disponível
   * Inserção de tabelas e imagens

6. **Salve como rascunho** ou **envie para aprovação**

Dicas para Criação
-----------------

* Use os templates predefinidos
* Mantenha linguagem clara e objetiva
* Inclua todas as informações necessárias
* Revise antes de enviar para aprovação
* Use formatação consistente

Como Aprovar um Documento
========================

Para Aprovadores
---------------

1. **Acesse "Aprovações" no menu**
2. **Visualize a lista de documentos pendentes**
3. **Clique no documento para revisar**
4. **Analise o conteúdo completo**
5. **Escolha uma ação:**
   * **Aprovar**: Documento será publicado
   * **Rejeitar**: Retorna para o autor com comentários
   * **Solicitar alteração**: Pede ajustes específicos

6. **Adicione comentários se necessário**
7. **Confirme a ação**

Critérios de Aprovação
--------------------

* Conformidade com normas
* Completude das informações
* Clareza e objetividade
* Adequação ao público-alvo
* Consistência com outros documentos

Como Registrar uma Não Conformidade
==================================

Identificação
------------

Uma NC deve ser registrada quando:

* Desvio de procedimento
* Produto/serviço não conforme
* Falha de equipamento
* Reclamação de cliente
* Auditoria interna/externa

Registro Passo a Passo
--------------------

1. **Acesse "Qualidade" > "Não Conformidades"**
2. **Clique em "Nova NC"**
3. **Preencha os campos:**
   * Descrição detalhada
   * Local de ocorrência
   * Data e hora
   * Responsável
   * Gravidade
   * Tipo de NC

4. **Anexe evidências** (fotos, documentos)
5. **Defina ações imediatas** (contenção)
6. **Salve o registro**

Acompanhamento
-------------

* NC será atribuída a um responsável
* Prazo para análise será definido
* Plano de ação será elaborado
* Implementação será monitorada
* Eficácia será verificada

Como Usar os Relatórios
=======================

Tipos de Relatórios
------------------

* **Dashboard Executivo**: Visão geral dos indicadores
* **Relatório de Documentos**: Status e estatísticas
* **Relatório de NCs**: Análise de não conformidades
* **Relatório de Equipamentos**: Status de calibração
* **Relatório de Auditorias**: Resultados e achados

Gerando Relatórios
-----------------

1. **Acesse "Relatórios"**
2. **Selecione o tipo desejado**
3. **Configure os filtros:**
   * Período
   * Setor
   * Status
   * Responsável

4. **Clique em "Gerar Relatório"**
5. **Visualize online** ou **exporte (PDF/Excel)**

Dicas de Uso
-----------

* Use filtros para análises específicas
* Agende relatórios recorrentes
* Compartilhe com stakeholders
* Acompanhe tendências
* Base para tomada de decisões

Troubleshooting
==============

Problemas Comuns
---------------

**Não consigo fazer login**
* Verifique usuário e senha
* Contate o administrador
* Use "Esqueci minha senha"

**Documento não carrega**
* Verifique sua conexão
* Recarregue a página
* Contate o suporte

**Erro ao salvar**
* Verifique campos obrigatórios
* Tamanho do arquivo (máx 10MB)
* Formato suportado

**Não recebo notificações**
* Verifique configurações do perfil
* Confira pasta de spam
* Atualize dados de contato
